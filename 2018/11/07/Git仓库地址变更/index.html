修改git仓库地址

发表于 2018-11-07 | 分类于 git

1.系统

windows：项目根目录下鼠标右键，点击git bash here 打开终端

Mac OS： terminal终端进入项目根目录

注：如果当前工程处于修改状态，请先git commit

2.查看当前远程仓库地址

git remote -v

origin  http://101.37.69.236:3000/TianjinIT/PacificLinksMall-Android (fetch)
origin  http://101.37.69.236:3000/TianjinIT/PacificLinksMall-Android (push)

3.删除远程仓库

git remote rm origin

4.修改远程仓库地址

git remote add origin [url]

将步骤2中得到的仓库地址中的 101.37.69.236 替换为 192.168.30.39

$ git remote add origin http://192.168.30.39:3000/TianjinIT/PacificLinksMall-Android

5.获取新仓库信息和分支情况

git pull

6.本地分支关联到远程分支

git push --set-upstream origin [分支名]

$ git push –set-upstream origin v2.5.0

7.远程仓库地址变更完成，可以愉快的提交代码了！！！
